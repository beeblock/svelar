# svelar

This package is the unscoped npm entrypoint for the Svelar CLI.

```bash
npx svelar new my-app
```

The framework implementation lives in [`@beeblock/svelar`](https://www.npmjs.com/package/@beeblock/svelar). This package only forwards the `svelar` binary so first-run scaffolding can use the short Laravel-style command.
